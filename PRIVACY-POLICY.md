## What data does this extension collect?

**None.** The AI Prompt Helper extension does not collect, store, transmit, or share any user data whatsoever.

## How it works

- All prompt improvement logic runs **locally in your browser**
- No data is sent to any external server
- No analytics or tracking of any kind
- No cookies are set
- The extension uses Chrome's `storage` API only to save your local preferences (like theme settings). This data never leaves your device.

## Permissions explained

| Permission | Why we need it |
|-----------|---------------|
| `activeTab` | To read text you've selected on the current page when you right-click |
| `contextMenus` | To add "Improve this prompt" to your right-click menu |
| `storage` | To save your preferences locally in your browser |

## Third-party services

This extension does **not** connect to any third-party services, APIs, or servers.

## Changes to this policy

If we ever change how this extension handles data (we don't plan to), we'll update this page and the extension listing.
